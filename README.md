## üìå What does this project do?

The **Corporate Carbon Footprint Tracker** analyzes corporate greenhouse gas (GHG) emissions using **Environment Data Analytics (EDA)** techniques.  
It focuses on:
- **Scope 1** (Direct emissions)
- **Scope 2** (Indirect energy emissions)
- **Scope 3** (Value chain emissions)

The project uses **Google Colab** and Python-based data analytics to explore emission patterns, trends, and major contributors.

- Useful for **academic learning, sustainability analysis, and research**
