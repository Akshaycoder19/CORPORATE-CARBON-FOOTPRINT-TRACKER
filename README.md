## ğŸ“– About the Project

The **Corporate Carbon Footprint Tracker** is an academic Environment Data Analytics project that focuses on analyzing and visualizing corporate greenhouse gas (GHG) emissions. The project studies emissions across **Scope 1, Scope 2, and Scope 3**, enabling a comprehensive understanding of how corporate activities impact the environment.

Using **Python and Google Colab**, this project applies **Exploratory Data Analysis (EDA)** techniques to uncover patterns, trends, correlations, and insights from emission datasets. The project aligns strictly with **Theme 2: Environment Data Analytics (EDA)** and **Task EDA-1**.

## ğŸ¯ Project Objectives

### Core Objective
- To analyze corporate carbon emission data covering **Scope 1, Scope 2, and Scope 3 emissions**.

### Data Analytics Objectives
- Classify companies based on **carbon intensity** (Low / Medium / High)  
- Perform **year-wise trend analysis** of emissions  
- Study the contribution of different emission scopes  
- Conceptually compare **actual emission trends vs pledged reductions**  
- Generate insights using data visualization techniques
